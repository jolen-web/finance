About Currency Settings

The currency setting controls which symbol is displayed throughout the application for all monetary amounts.

  • This setting is stored locally and persists across sessions
  • Changing the currency only updates the display symbol, not the actual values
  • All amounts remain in their original numeric values
  • For true multi-currency support with conversion, that would be a future enhancement
